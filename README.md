# 📊 Analisador de Produtividade Escolar

Um aplicativo interativo desenvolvido em Python para ajudar estudantes a monitorar, analisar e visualizar suas horas de dedicação aos estudos ao longo do tempo. O projeto gera gráficos evolutivos e armazena os dados de desempenho localmente.

## 🚀 Funcionalidades

- **Registro de Dados**: Interface simples para cadastrar datas e horas dedicadas aos estudos.
- **Análise Estatística**: Processamento automático dos dados para exibir métricas relevantes.
- **Visualização Gráfica**: Geração de gráficos de linha customizados mostrando a evolução das horas de estudo diárias.
- **Armazenamento Local**: Dados salvos e organizados de forma estruturada em um arquivo CSV.

## 🛠️ Tecnologias Utilizadas

- **Python**: Linguagem base do projeto.
- **Streamlit**: Para a criação da interface web interativa de forma rápida.
- **Pandas**: Para a manipulação, leitura e estruturação dos dados de produtividade.
- **Matplotlib**: Para a plotagem e customização do gráfico evolutivo.
